# Мой Список (MyList)

**Мой Список** — это современный персональный менеджер контента, разработанный для тех, кто ценит минимализм, структуру и эстетику. Приложение позволяет организовывать ваши планы на просмотр фильмов, чтение книг или изучение курсов в удобном и визуально приятном формате.

---

## ✨ Особенности дизайна и UX
- **Эстетика Black & White**: Глубокий черный фон с чистыми белыми акцентами для максимальной концентрации на контенте.
- **Интуитивное управление**:
  - Карточки категорий с мягкими градиентами, которые плавно исчезают, не перегружая интерфейс.
  - Динамическая цветовая гамма: каждая категория имеет свой цвет, который становится темой оформления при входе в неё.
  - Эргономичные кнопки управления: увеличенные FAB с оптимизированными отступами для удобного использования одной рукой.
- **Полный CRUD**: Редактирование и удаление категорий и элементов через единые, функциональные диалоговые окна.
- **Умная сортировка**: Быстрое переключение порядка (А-Я / Я-А) с наглядной индикацией.

## 🛠 Технологический стек

### Архитектура
Приложение построено на принципах **Clean Architecture** и разделено на два независимых Gradle-модуля:
- `:app`: Слой презентации (UI и ViewModels).
- `:core`: Слой данных и домена (Бизнес-логика, Use Cases, Репозитории, Room DB).

### База данных
Используется **Room Persistence Library** с поддержкой:
- Автоматического подсчета элементов в категориях через подзапросы.
- Каскадного удаления данных.
- Реактивных потоков данных через **Kotlin Flow**.

### Инъекция зависимостей
Весь проект интегрирован с **Hilt (Dagger)**, что обеспечивает слабую связность компонентов и упрощает тестирование.

### Фоновые задачи
- **WorkManager**: Реализована периодическая задача очистки/синхронизации с системными ограничениями (выполнение только при достаточном заряде батареи).
- **Android Service**: Фоновый сервис для мониторинга состояния приложения.

### Анимации и UI
Интерфейс полностью реализован на **Jetpack Compose** с использованием:
- `animateContentSize` для плавного изменения размеров карточек.
- `AnimatedVisibility` для динамических элементов интерфейса (например, поиск).
- Кастомных переходов между состояниями вкладок.

### Интеграция технологий
В приложении реализован гибридный подход: экран «О приложении» использует **XML-разметку**, в которую встроен **ComposeView**, демонстрируя возможности бесшовной интеграции нового и старого стеков разработки.

### Конфигурация сборки
Настроены различные варианты поставки приложения:
- **Product Flavors**: `demo` (ознакомительная) и `full` (полнофункциональная) версии.
- **Build Types**: Оптимизированные `debug` и `release` сборки с настроенным **ProGuard/R8** для защиты и сжатия кода.
